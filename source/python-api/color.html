Color¶

class Color(r: int = 0, g: int = 0, b: int = 0, a: int = 255, color_space: iterm2.color.ColorSpace = <ColorSpace.SRGB: 'sRGB'>)¶

Describes a color.

Parameters:
  • r – Red, in 0-255
  • g – Green, in 0-255
  • b – Blue, in 0-255
  • a – Alpha, in 0-255
  • color_space – The color space. Only sRGB is supported currently.
