Tmux¶

Represents tmux integration objects.

async_get_tmux_connections(connection)¶

Fetches a list of tmux connections.

Parameters:connection – An existing iterm2.Connection.
Returns:A list of TmuxConnection objects.
class TmuxException¶

A problem was encountered in a Tmux request.

class TmuxConnection(connection_id, owning_session_id, app)¶
async_send_command(command)¶

Sends a command to the tmux server.

Parameters:command – The command to send to tmux (e.g., “list-sessions”)
Returns:The command’s output as a string.
Throws TmuxException:
 If the command fails for any reason or the RPC fails such as from an invalid ID.
async_set_tmux_window_visible(tmux_window_id, visible)¶

Hides or shows a tmux window.

Tmux windows are represented as tabs in iTerm2. You can get a tmux_window_id from iterm2.Tab.tmux_window_id(). If this tab is attached to a tmux session, then it may be hidden. Use iterm2.tmux.async_open_window() to make it visible again.

Parameters:
  • tmux_window_id – The window to show or hide.
  • visible – True to show a window, False to hide a window.
connection_id¶

Returns the unique identifier of the connection.

Returns:A connection ID (a string), which is also a human-readable description of the connection.
owning_session¶

Returns the “gateway” session.

Returns:The Session where tmux -CC was run, or None if it cannot be found.
