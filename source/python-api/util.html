Utilities¶

Provides handy functions.

frame_str(frame)¶

Formats an api_pb2.Frame or Frame as a human-readable string.

Parameters:frame – An api_pb2.Frame or Frame
Returns:A human-readable string.
size_str(size)¶

Formats an api_pb2.Size or Size as a human-readable string.

Parameters:frame – An api_pb2.Size or Size:
Returns:A human-readable string.
class Size(width, height)¶

Describes a 2D size.

Can be used where api_pb2.Size is accepted.

json¶

Gives a JSON representation of the size.

class Point(x, y)¶

Describes a 2D coordinate.

Can be used where api_pb2.Point is accepted.

json¶

Returns a JSON representation of the point.

class Frame(origin=(0, 0), size=<iterm2.util.Size object>)¶

Describes a bounding rectangle. 0,0 is the bottom left coordinate.

json¶

Returns a JSON representation of the frame.

origin¶

The top-left coordinate.

Returns:A Point.
size¶

The size.

Returns:A Size.
class CoordRange(start, end)¶

Describes a range of contiguous cells.

Parameters:
  • start – A Point giving the start point.
  • end – A Point giving the first point after the start point not in the range.
end¶
Returns:The first Point after self.start not in the range.
start¶
Returns:The start Point.
class Range(location, length)¶

Describes a range of integers.

Parameters:
  • location – The first value in the range.
  • length – The number of values in the range.
length¶
Returns:The length of the range.
location¶
Returns:The first location of the range.
class WindowedCoordRange(coordRange, columnRange=None)¶

Describes a range of coordinates, optionally constrained to a continugous range of columns.

Parameters:
  • coordRange – The CoordRange of cells.
  • columnRange – The Range of columns, or None if unwindowed.
columnRange¶
Returns:The range of columns, a Range, or an empty range if unconstrained.
coordRange¶
Returns:The range of coordinates, a CoordRange.
