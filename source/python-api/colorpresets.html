Color Presets¶

Provides access to color presets.

class ColorPreset(proto)¶
static async_get(connection, name)¶

Fetches a color preset with the given name.

Parameters:
  • connection – An iterm2.Connection.
  • name – The name of the preset to fetch.
Returns:

Either a new ColorPreset or None.
